Adding a Custom Home Screen

In this guide, we'll show you how to set a custom home screen for Sneakon by editing the builderdata.pbcfg file.

Steps to Add a Custom Home Screen

  1. Locate the builderdata.pbcfg file in the Sneakon installation directory.
  2. Open the builderdata.pbcfg file using a text editor like Notepad.
  3. The entire file should only contain the URL. For example:
    4. 
http://example-default-homepage-url.com
  4. Replace the existing URL with the URL of your desired homepage. For example:
    5. 
http://your-custom-homepage.com
  5. Save the changes and close the text editor.
  6. Restart Sneakon to apply the changes.

Congratulations! You've successfully set a custom home screen for Sneakon.
